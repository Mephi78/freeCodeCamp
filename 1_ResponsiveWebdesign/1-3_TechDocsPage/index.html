Grundlagen

JavaScript wurde 1995 von Netscape entwickelt. Die Skriptsprache ist auch bekannt als JS oder offiziell ECMAScript2019/ES2019.

JavaScript ist case sensitive. Namenskonvention: Camel Case (erstes Wort lower case, weitere Wörter starten mit upper case).

// Das ist ein einzeiliger Kommentar
/* mehrzeiliger Kommentar*/

Variablen

Variablennamen können aus Buchstaben, Ziffern, $ und _ bestehen, aber nicht mit einer Ziffer beginnen.
Variablen werden deklariert mittels var, const oder let und initialisiert mit dem Zuweisungsoperator =.
Konstanten müssen initialisiert werden mit const konstante = value; und sind nicht änderbar.

typeof my_string gibt den Typ der Variablen my_string zurück, bspw. String.

  • var deklariert eine globale Variable; kann redeklariert werden; global scope
  • let deklariert eine lokale Variable im Scope eines Blocks; block scope, nicht erneut deklarierbar
  • const deklariert eine nur lesbare Konstante im Scope eines Blocks und initialisiert sie einmalig
  • Faustregel: const immer, wenn ein Wert oder ein Datentyp sich nicht ändern soll, sonst let
  • const immer für new Array, Object, Function, RegExp

Datentypen

Nach ECMA-Standard gibt es acht Datentypen, die primitiven Typen String, Number, Boolean, undefined, null, Bigint, Symbol und das komplexere Object.

Strings werden verwendet, um Text darzustellen. Sie werden unter Verwendung von einfachen oder doppelten Anführungszeichen deklariert.
Das String-Objekt hat verschiedene Methoden, z.B. substring(), charAt(), toUpperCase().

Operatoren

  • Zuweisungsoperator: =
  • normaler == und strikter Vergleichsoperator === (selber Wert UND Typ), Ungleichheit !=
  • Arithmetische Operatoren (Addition, Subtraktion, Multiplikation, Division): + - * /
  • Modulo (Rest), Exponent: % **
  • Vergleichsoperator (gleich, ungleich, größer, kleiner): == != > <

Kurzschreibweise: x = x + y ist bspw. dasselbe wie x += y

Operatoren sorgen für eine automatische Typenkonvertierung, z.B. console.log("5" - 1) // 4

Math Object

Das Math-Objekt enthält statische Eigenschaften und Methoden für mathematische Konstanten und Funktionen.

  • Math.random() gibt einen Zufallswert >= 0 und < 1 zurück
  • Math.floor() rundet eine Fließkommazahl ab
  • Math.ceil() rundet eine Fließkommazahl auf
  • Bsp.: Math.floor(Math.random() * 6) + 1 gibt eine Zufallszahl von 1 bis 6 zurück (Würfel)
